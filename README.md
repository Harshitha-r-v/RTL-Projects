# RTL Projects

A collection of Register Transfer Level (RTL) design projects focused on digital hardware design, ASIC implementation, FPGA prototyping, and hardware validation. These projects demonstrate hands-on experience in Verilog/SystemVerilog, digital design methodologies, and complete hardware development flows from specification to implementation.

## Overview

This repository contains RTL-based projects developed as part of academic coursework, research, and independent work. The designs emphasize:

- Digital system design and microarchitecture development
- RTL coding using Verilog/SystemVerilog
- Functional verification and waveform debugging
- FPGA prototyping and hardware validation
- ASIC design flow including synthesis, STA, and physical design
- Performance and power optimization techniques

## Skills Demonstrated

- RTL Design using Verilog/SystemVerilog
- FSM Design and Datapath Architecture
- FPGA Prototyping on Xilinx Artix-7
- Static Timing Analysis (STA) and Timing Closure
- Clock Domain Crossing (CDC) Concepts
- SDC Constraints and Timing Debug
- ASIC Flow: Synthesis → Floorplanning → CTS → Routing → PDN
- Hardware Debug and Validation
- Python-based Automation for Testing
